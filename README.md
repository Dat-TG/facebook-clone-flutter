# Flutter Facebook Clone. 

Flutter Facebook Clone. 
This is a simple flutter application where I have clonned Facebook UI.

## Installation

``` 

git clone https://github.com/Dat-TG/facebook-clone-ui.git
flutter pub get
flutter run 

```
